# LumiNove — Shopify Theme

Custom Shopify theme (Horizon-based) for [LumiNove](https://luminova-shop-2.myshopify.com), a decorative lighting store targeting the Spanish market.

## Structure
- `sections/` — theme sections
- `blocks/` — reusable content blocks
- `snippets/` — Liquid snippets
- `templates/` — page/product/collection templates
- `assets/` — CSS, JS, images
- `locales/` — translations (primary: es)
- `config/` — theme settings schema and data

## Deployment
This repo is connected to the Shopify Admin theme editor via the GitHub integration. Changes pushed to `main` sync to the `lumi-theme` theme.

## Brand
- Primary color: `#C9A84C` (gold)
- Dark scheme: `#0a0a0a`
- Background: `#f5f0e8`
